# jusmore-ev

This project provides EV charging station details in real time. The useful details like status of availability of charging stations, waiting period, facilities for recreation, trip planning, mode of payment, map view, etc.

## React Native Expo App

This is a React Native Expo project created with the default template.

## Getting Started

### Prerequisites

- Node.js (v20 or higher recommended)
- npm or yarn
- Expo Go app on your mobile device (for testing)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:

```bash
npm start
```

This will start the Expo development server. You can then:

- **Run on Android**: Press `a` in the terminal or run `npm run android`
- **Run on iOS**: Press `i` in the terminal or run `npm run ios` (macOS only)
- **Run on Web**: Press `w` in the terminal or run `npm run web`
- **Use Expo Go**: Scan the QR code with the Expo Go app on your mobile device
